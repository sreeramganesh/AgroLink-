# 🌾 AGROLINK – Farm to Market Platform

AGROLINK is a full-stack agriculture marketplace that connects farmers directly with buyers, eliminating intermediaries. The platform enables farmers to list products and buyers to purchase them efficiently with AI-powered insights.

---

## 🚀 Features

- 👨‍🌾 Farmer Dashboard
  - Add and manage products
  - Upload product images
  - View orders and earnings

- 🛒 Buyer Dashboard
  - Browse available products
  - View product details
  - Purchase directly from farmers

- 🔐 Authentication
  - Secure login/signup using Supabase Auth
  - Role-based access (Farmer / Buyer)

- 🗄️ Database Integration
  - Supabase (PostgreSQL)
  - Tables: Users, Products, Orders

- 🖼️ Image Upload
  - Supabase Storage integration
  - Public image access

- 🧠 AI Features
  - Demand prediction
  - Price suggestion
  - Smart farmer matching

---

## 🛠️ Tech Stack

- Frontend: Next.js, Tailwind CSS  
- Backend: Next.js API Routes  
- Database: Supabase (PostgreSQL)  
- Authentication: Supabase Auth  
- Storage: Supabase Storage
